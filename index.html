oPlayer Themes

Community-built visual themes for oPlayer. Download a .zip, import it in the app, and transform your player.

How to Install

  1. Download the .zip file (do not unzip)
  2. Open oPlayer → Settings → Themes
  3. Turn on Enable External Themes
  4. Tap Import New Theme and select the .zip
  5. Select your new theme from the list to apply

Want to build your own theme? oPlayer themes are standard HTML, CSS, and JavaScript.
